What is Droplets?
It is a micro savings apps that identify small savings opportunities within your existing budget and automatically sweep money into a savings account. They amounts are small â€” sometimes cents at a time.

In this application we have implemented essential backend microservices such user microservice, Goals microservice, Transaction microservice, Rules microservice, Transaction microservice, Notification microservice.


## How to run the project!
### - Clone the project
> ``` git clone  "https://github.com/gsunil2502/Droplets" ```

> ```cd Droplets```

 ### - Requirements
 
 > ```Java```
 > ```Spring Boot Application```
 > ```maven```
 > ```PostgresSql```
 > ```Spring JPA```
 > ```Spring JWT```
 > ```Post Man Application```
